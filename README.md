## 🌟 Visão Geral | Overview

**Judeus Verdadeiros** é um projeto dedicado à exploração das raízes bíblicas e históricas da fé. Este repositório contém o código-fonte do site oficial, projetado com uma estética premium, moderna e bilingue.

> "A busca pela luz em um mundo de sombras."

---

## ✨ Funcionalidades | Features

### 🎨 Estética & Design
- **Glassmorphism UI**: Uma interface moderna com elementos translúcidos e desfoque de fundo.
- **Dark & Light Mode**: Suporte nativo para temas claro e escuro, respeitando a preferência do usuário.
- **Special Effects**: Efeitos de iluminação interativos (glow) e animações de scroll suaves.

### 🌍 Acessibilidade & Internacionalização
- **Bilingual (PT/EN)**: Tradução completa entre Português e Inglês com troca instantânea.
- **Responsividade**: Experiência fluida em qualquer dispositivo, do mobile ao desktop.
- **Acessibilidade (a11y)**: Escrito com HTML semântico e práticas recomendadas para leitores de tela.

### ⚡ Performance
- **Zero Dependências Pesadas**: Construído puramente com HTML5, CSS3 e JavaScript Moderno.
- **Otimização de Ativos**: Carregamento rápido e transições leves.

## 🚀 Como Executar | How to Run

1. **Clone o repositório**:
   ```bash
   git clone https://github.com/judeusverdadeiros/judeusverdadeiros.github.io.git
   ```
2. **Abra o arquivo `index.html`** no seu navegador ou use uma extensão como o *Live Server*.
